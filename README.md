# CSC482-FinalGroupProject

## Downloading Embeddings: 
-  
`python3 embeddings.py`

## Training All Models: 
- To run with k-fold: 
`python3 sentiment_analysis.py -train -k`
- To run without k-fold: 
`python3 sentiment_analysis.py -train`

## Testing All Models: 
- Command: 
`python3 sentiment_analysis.py -test`
